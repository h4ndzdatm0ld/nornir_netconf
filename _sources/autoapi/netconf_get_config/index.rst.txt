.. py:function:: netconf_get_config(task: nornir.core.task.Task, source: Optional[str] = 'running', path: Optional[str] = '', filter_type: Optional[str] = 'xpath') -> nornir.core.task.Result

   Get configuration over Netconf from device.

   :param source: Configuration datastore to collect from. Defaults to `running`
   :type source: Optional[str]
   :param path: Subtree or xpath to filter. Defaults to `''`
   :type path: Optional[str]
   :param filter_type: Type of filtering to use, 'xpath' or 'subtree'. Defaults to `xpath`
   :type filter_type: Optional[str]

   .. rubric:: Examples

   Simple example::

       > nr.run(task=netconf_get_config)

   Collect startup config::

       > nr.run(task=netconf_get_config, source="startup")


   Passing options using ``xpath``::

       > xpath = "/devices/device"
       > nr.run(task=netconf_get_config,
       >        path=xpath)

   Passing options using ``subtree``::

       > subtree = "<interfaces></interface>"
       > nr.run(task=netconf_get_config,
       >        filter_type="subtree",
       >        path=subtree)

   :returns:

             Result object with the following attributes set::

                 * result (RpcResult): Rpc and Manager
